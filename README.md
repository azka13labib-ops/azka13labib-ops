## About Me
Saya Azka Labib, fokus di **Front-end** dan **UI/UX**. Saya suka bikin tampilan yang **rapi**, **responsif**, dan **interaktif** dengan animasi halus, micro-interactions, dan struktur code yang gampang dirawat.

**Yang saya kejar:**
- UI yang clean (spacing, hierarchy, grid rapi)
- UX yang cepat dan enak dipakai (performance + aksesibilitas)
- Motion yang subtle (tidak berlebihan, tapi terasa premium)

**Highlight:**
- Juara 1 lomba web design ITB Widyagama Lumajang (project WG38)
